2015 Odesk Разработка "хитрой" системы управления пользователями
Functional Specification
- A website which will allow a user to create a profiles containing content.
- An admin can have multiple profiles

For each profile:
- Each profile will have a public URL that can be viewed by anyone
- Profiles can be edited by the admin online (simple text fields for now will be sufficient)
- Very basic look and feel, this will be designed later.
- Each profile will also have an extended profile, which is accessed via a passcode
- The passcodes are generated by the admin
- New passcodes can be sent to an email address
- There can be multiple passcodes for extended profiles
- Individual Passcodes can be de-activated
